# 📘 Gemma 3:1b 최적화 수학 문제 프롬프트 세트

이 문서는 Google의 Gemma 3:1b 언어모델에 최적화된 수학 문제 프롬프트 모음입니다.
실수를 줄이고 정답률을 높이기 위한 구조적 지침을 포함하고 있습니다.

---

## 🧩 1. 기본 1차 방정식 프롬프트

```
문제: 2x + 3 = 11

지침:
1. 방정식을 왼쪽과 오른쪽으로 나누고 구조를 이해하라.
2. 항을 정리하고, 같은 종류의 항끼리 모아라.
3. x에 대해 식을 정리하라.
4. x의 값을 구하라.
5. 구한 x값을 원래 방정식에 대입하여 검산하라.
6. 최종 정답은 반드시 "x = [정답]" 형태로 출력하라.
```

---

## 🧮 2. 분수가 포함된 방정식

```
문제: (3x - 1)/2 = (x + 5)/4

지침:
1. 양변에 최소공배수를 곱하여 분모를 제거하라.
2. 괄호를 풀고 정리하라.
3. 항을 정리하고 x를 한쪽으로 모아라.
4. x에 대한 해를 구하라.
5. 구한 값을 원래 식에 대입하여 좌변 = 우변이 되는지 검산하라.
6. 정답은 "x = [정답]" 형식으로 끝에 한 줄만 출력하라.
```

---

## 🧠 3. 이차 방정식

```
문제: x^2 - 5x + 6 = 0

지침:
1. 이차 방정식의 해법(인수분해, 근의 공식 등) 중 하나를 선택하라.
2. 선택한 방식에 따라 단계를 나눠 풀이하라.
3. 해를 두 개 구했다면 각각 검산하라.
4. 정답은 "x = 2 또는 x = 3" 형식으로 한 줄로 정리하라.
```

---

## 🔄 4. 검산 강제 프롬프트

```
문제: 4(x - 2) = 3x + 6

지침:
1. 단계적으로 풀어라(미출력)
2. 마지막에는 반드시 구한 해를 원래 식에 대입해서 좌변과 우변이 같은지 비교하라.(미출력)
3. 계산이 틀렸을 경우, 반드시 다시 풀고 정확한 해를 구하라.
4. 검산이 성공했을 경우, 지침 1,2번 과정을 자세하게 알려주고 정답은 간단하게 정리하라




"x = ?" 형식으로 단정적으로 말해라.


고, 그 이유도 덧붙여라.
```

---

## 🧪 5. 단계 출력 형식 통제

```
다음 수학 문제를 해결하라:

문제: 3(x + 2) - 5 = 2x + 7

출력 형식:
[1단계] 괄호 풀기: ...
[2단계] 항 정리: ...
[3단계] x 모으기: ...
[4단계] 해 구하기: ...
[5단계] 검산: ...
[정답] x = ...
```

---

## 🧠 팁: Gemma의 수학 정확도를 높이는 법

- 계산 실수를 줄이기 위해 단계별 풀이를 강제하라.
- 해를 구한 후 반드시 검산하도록 유도하라.
- 출력 형식을 지정하여 혼동을 막아라.
- 분수/소수 문제에서는 최소공배수와 괄호 정리를 명확히 하라.




문제: x^4-5x^2+4=0
지침:
1. 단계적으로 풀는데 나한테 알려주지마
2. 마지막에는 반드시 구한 해를 원래 식에 대입해서 좌변과 우변이 같은지 비교하는데 나한테 알려주지 마
3. 계산이 틀렸을 경우, 반드시 다시 풀고 정확한 해를 구하라.
4. 검산이 성공했을 경우, 지침 1,2번 과정을 자세하게 알려주고 모든 x의 해를 간단하게 정리하라

정답은 간단하게 정리하라
